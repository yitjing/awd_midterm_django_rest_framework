Advanced Web development Midterm Rest API

This page is for introducing the midterm assignment to implement Django REST API

Rest Specification

All of the required rest specification will be listed below:

GET Routes

  • api/protein/A0A016S8J7
    (api/protein/[PROTEIN ID]) Return the protein sequence and all we know about it
  • api/pfam/PF00360
    (api/pfam/[PFAM ID]) Return the domain and it's description
  • api/proteins/55661
    (api/proteins/[TAXA ID]) Return a list of all proteins for a given organism
  • api/pfams/55661
    (api/pfams/[TAXA ID]) Return a list of all domains in all the proteins for a given organism.
  • api/coverage/A0A016S8J7
    (api/coverage/[PROTEIN ID]) Return the domain coverage for a given protein.

Post Routes

  • api/protein/
    (api/protein/) Add a new record
